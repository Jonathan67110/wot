2. Moteur

a. Depuis le wiki
- Aller à l'adresse suivante : http://wiki.worldoftanks.com/Engine
- Ouvrir un tableur et coller la liste complète des moteurs, en prenant bien l'en-tête de toutes les nations
- Après la colonne "price", ajouter une colonne type_credit : Pour chaque ligne ayant un crédit d'achat du moteur non-nul, ajouté le caractère 'c'
Possibilité de faire un test (Si(case<>""; alors 'c', sinon, '')
- sauvegarder
- Après la colonne "weight", ajouter une colonne "pays" : Pour chaque liste, mettre le nom du pays, comme ci-après:
* American engine 	: mettre 'USA'
* British Engines 	: Mettre 'Royaumes-Unis'
* German Engines 	: Mettre 'Allemagne'
* French Engines 	: Mettre 'France'
* Soviet Engines 	: Mettre 'Russie'
* Chinese Engines 	: Mettre 'Chine'
- sauvegarder
- Déplacer la colonne 'Price' et 'Type_credit' après la colonne 'compatibility'
- sauvegarder
- Déplacer la colonne 'Name' en première colonne
- sauvegarder
- Déplacer la colonne 'Tier' après la colonne 'weight'
- sauvegarder
- Supprimer les lignes vides, 
- Supprimer les lignes contenant 'American engine',  'British engine', 'German engine', 'French engine', 'Soviet engine', 'Chinese engine'
- Supprimer toutes lignes contenant des en-têtes de colonnes SAUF la première ligne
- sauvegarder
- Renommer les noms des en-têtes:
* Name => Nom
* Power => Puissance
* Type => Type_moteur
* Chance of Fire => Chance_incendie
* Weight => Poids
* Tier => Tier_latin
* Compatibility => Compatibilite
* Price => Prix
- Ajouter un en-tête 'expérience', après 'Type_credit'

b. Complément sur le site de wot

- Aller sur le site : http://worldoftanks.eu/encyclopedia/vehicles/
- Copier dans le tableur, pour chaque tank japonais, les caractéristiques du moteur, en vérifiant qu'il ne se retrouve pas sur un char précédent. 
Les données peuvent être coller à la suite, en laissant une ligne libre. A chaque copie, ajouter le nom du char en fin de ligne, pour chaque moteur copié.
- Supprimer les lignes vides qui ont été ajoutées,
- Vérifier que pour chaque moteur, le nom d'un char est bien assigné
- Déplacer le prix dans la colonne prix
- Ajouter le caractère 'c' pour chaque montant non vide (= élément de char premium) Possibilité de faire un test (Si(case<>""; alors 'c', sinon, '')
- sauvegarder
- Déplacer le nom des chars dans la colonne compatibilite
- Déplacer le tier dans la colonne tier_latin
- Déplacer le nom du moteur dans la colonne nom
- Déplacer la colonne puissance (3ème colonne) dans la colonne puissance, en même temps que la colonne chance_incendie
- Vérifier les noms des moteurs pour voir si il y a doublon. En cas de doublon, ajouté le nom d'un char à la suite du char du moteur identique, puis, supprimer la ligne du char copié.

Le nom du char ajouté devra respecter la convention suivante : sépration entre chaque char par les caractères ', ' (virgule puis espace)
possibilité de faire un tri sur le nom des chars pour facilité la manoeuvre
- Pour chaque moteur, chercher le poids du moteur dans le jeu WoT (r). Inutile de rajouter l'unité, uniquement le poids en kilo est souhaité
- Pour le type de moteur, voici ce que les recherches donne :
* Moteur associé au Chi-ri : moteur essence
* Autre moteur : diesel
- Ajouter 'Japon' au pays du moteur
- Supprimer les lignes, s'il reste un espace entre la dernière ligne des données précédentes et les lignes des nouvelles données : le tableau ne doit contenir à présent aucune ligne vide
- Sauvegarder et fermer le fichier

X. Création des tableaux d'appartenance des éléments aux chars 

1. Partir du fichier CSV fabriqué, pour chaque élément (tourelle, canon, moteur, radio, suspension, ...), et exécuter les actions suivantes:

- Faire une copie exacte du fichier. Sauvegarder la copie sous 'Lien_tank_element' ou element est le nom de l'élément à lier aux tank.
- Ouvrir le fichier sous un tableur
- Supprimer toutes les colonnes, sauf, les colonnes nom de l'élément, pays et compatibilite
- Remplacer le caractère ',' par le caractère '!' dans la colonne 'nom' de l'élément
- Renommer l'en-tête de colonne 'compatibilite' en '1'
- ajouter des en-tête de colonne, nommé de 2 à 20;
- sauvegardez en .csv, format séparateur = ';'
- Fermer le fichier
- Ouvrir le fichier avec un éditeur, type bloc note ou autre, et remplacer exactement le caractère ', ', en copiant ce caractère ou en exécutant la commande sous alt+f, si la conversion ultérieure ne marche pas. Si aucune conversion demandée ci après ne s'éxecute, changer de logiciel.
	Remplacement à effectuer : remplacer le caractère ', ' exactement par ';'. Il ne doit plus y avoir d'espace après le ';'
- Effectuer plusieurs itérations de remplacement de la suite de caractère ';;' par ';' pour éviter les répétitions multiples de ';'
- Sauvegarder une dernière fois
- Ouvrir le fichier sous un tableau
- Sélectionner la première colonne et remplacer '!' par ',', puis, sauvegarder
- vérifier si la conversion a marché, et, si tous les types de véhicules ont bien été séparés

a. Cas du fichier Lien_tank_moteur
- les données suivantes devront être reconcaténées:
* "Twin Cadillac" et "Series 44T4" 
* "Twin Cadillac" et "Series 42"
* Les char suivant devront être renommé : 
	*WZ-111' doit être renommé en 'WZ-111 1-4'
	*AMX 50 B" en "AMX 50B"
	* "AMX 12 t" -> "AMX 12t"
	* "Tiger II" -> "Pz.Kpfw. Tiger II"
Malgré les précautions précédentes, ils risquent d'être séparés.
- Les caractères suivants devront tous être modifiés dans le tableur ou l'éditeur de texte : 
* "ä" -> "a"
* "ö" -> "o"
* "é" -> "e"

- Les caractères suivants devront tous être modifiés dans le tableur ou l'éditeur de texte :

XI. Nom des chars dans le fichier 'tank.csv'

Les chars suivant seront à vérifier et devront suivre la nomenclature suivante:
- Les chars français suivants devront être renommé :
* type AMX devront avoir un espace qui suit le terme AMX si il est suivi d'un caractère
* "AMX12 t" -> "AMX 12t"
* "BDR G1 B" -> "BDR G1B "
* "S35 CA" -> "S 35 CA"
* "AMX50 120" -> "AMX 50 120"
* "AMX AC mle. 46" -> "AMX AC Mle. 1946"
* "AMX AC mle. 46" -> "AMX AC Mle. 1948"
* "AMX 50 B" -> "AMX 50B"
- La série de char E25, E50, E75, E-50 Ausf. M et E100 devrait être écrit avec un tiret entre le E et le chiffre: E-25, E-50, E75, E100, E-50 Ausf. M
- Les chasseurs de chars allemands suivant devront être renommé: 
* "Jagdpanzer IV" -> "JagdPz IV"
* "Jagdpanzer E 100" -> "JagdPz E-100"
* "Panzerjäger I" -> "Panzerjager I" 

- La série des chars allemands citée ci-après devra respecter la forme suivante :
* "Tiger II" -> "Pz.Kpfw. Tiger II"
* "Pz.Kpfw. V/IV" -> "Pz.Kpfw. V/IV Alpha"
* "Tiger I" -> "Pz.Kpfw. VI Tiger"
* "Panther" -> "Pz.Kpfw. V Panther"
* "Tiger (P)" -> "Pz.Kpfw. VI Tiger (P)"

- Le char américain devra être renommé :
* "T26E4 SuperPershing" -> "T26E4 Super Pershing"
